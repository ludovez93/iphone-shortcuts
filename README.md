# Spese Tracker - iPhone Shortcut

## Installazione (1 minuto)

### Passo 1: Scarica
- Tap su `Spese.shortcut` qui sopra
- Tap **Download**
- Si apre **Comandi Rapidi** → tap **Aggiungi comando rapido**

### Passo 2: Collega ad Apple Pay (4 tap)
1. Apri app **Comandi Rapidi**
2. Vai su tab **Automazioni** (in basso)
3. Tap **+** → scegli **Transazione**
4. Seleziona le tue carte → **Avanti**
5. Tap **Esegui comando rapido** → scegli **Spese**
6. **IMPORTANTE**: disattiva "Chiedi prima di eseguire"
7. Tap **Fine**

### Fatto!

---

## Come funziona

| Situazione | Cosa succede |
|------------|-------------|
| Paghi con Apple Pay | Lo shortcut parte DA SOLO, registra tutto automaticamente |
| Vuoi aggiungere una spesa manuale | Apri lo shortcut dalla home, ti chiede esercente e importo |

Ogni spesa ti chiede la categoria: Cibo, Trasporti, Abbigliamento, Svago, Bollette, Altro.

## Dove trovo le spese?

Apri l'app **File** su iPhone → **iCloud Drive** → cartella **Shortcuts** → file **Spese.csv**

Il file si apre con Numbers o Excel e contiene: Data, Ora, Esercente, Categoria, Importo.
